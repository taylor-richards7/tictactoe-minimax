# tictactoe-minimax

A simple Tic Tac Toe game in Python with a Minimax AI ( single-player vs computer )

# What this project demonstrates
- Basic 2D game logic using a 1–9 board mapping
- Win / draw detection
- Recursion (Minimax) to search future game states
- Choosing the best move by maximising (AI) and minimising (player)

# How Minimax works ( short explanation )
Minimax searches all possible future moves:
- If the AI can force a win: score = +1
- If the player can force a win: score = -1
- If it ends in a draw: score = 0

The AI chooses the move that leads to the highest score assuming the player also plays most optimally
